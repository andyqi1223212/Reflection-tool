## 7. 常见困惑 FAQ

**Q：`agents/` 和 `agent第二轮/` 都有 `pipeline-b-style.prompt.md`，改哪个？**  
A：**只改 `agent第二轮/`**。`agents/` 已归档到 `archive/2026-05-23-cleanup/_old-agents/`。

**Q：lexicon 的版本号怎么走？**  
A：单调递增。当前 `context/pipeline-b-style-lexicon-v3.md`；每次 review apply 自动 bump v3 → v4，老版进 `context/_archive/`。

**Q：我应该改 `chains.json` 还是 v3 md？**  
A：**只改 v3 md**。`chains.json` 是 export 产物，下次 export 会覆盖。

**Q：dogfood 反馈数据在哪？**  
A：`users/qihaoyu/feedback.jsonl`（HTTP 同源 POST 写入）；本机文件，gitignored 与否看 `.gitignore`。

**Q：单独跑 B 一次试新 lexicon 怎么做？**  
A：
```bash
./venv/bin/python3 -c "
from agents_runtime.agents import run_b
import json
with open('runs/<run_id>/a.json') as f: a = json.load(f)
print(run_b(a))
"
```
